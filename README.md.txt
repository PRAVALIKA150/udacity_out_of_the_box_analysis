# Out of the Box Analytics by Pravalika

This Power BI project analyzes advertising performance to help Metro Staffing optimize monthly ad spend.

## KPIs Analyzed
- **KPI 1 – Spend Efficiency:** Spend trends over time, by category, and by vendor
- **KPI 2 – Vendor Performance:** Vendor-wise spend concentration and cost efficiency
- **KPI 3 – Campaign Performance:** Clicks, applications, CPC, and CPA by job category

## Executive Summary
An executive dashboard combines insights from all KPIs and recommends a predictive analytics approach (budget optimization / regression analysis) to improve ROI.

## Tools Used
- Power BI Desktop
- Data Visualization
- KPI Analysis
